CodeWriter Data Files

January 16, 2000



128K Trap Macros

  These are the 68K traps found in the 128K ROMs introduced with the 512Ke and the Mac Plus.  The reason for limiting the data file (and your code) to these traps is for backwards compatibility.  If you want maximum  backward compatibility back to the original 128K and 512K Macs, don’t use the traps in Inside Macintosh volume IV.  Basically these are the HFS traps and a handful of others (no zooming windows!).


256K+ Traps

  These are as many of the later 68K traps as I had the time/information/inclination to include.


A-Trap List

  All the A-traps that I know about.  Gleaned from many sources.  There are some blanks.  Until someone clues me in they will remain nameless.


C Routines

  A few C routines.  By popular demand.  Well, Ken McLeod asked for them, but that was before he worked for Apple.  And he still had a beard then.  Sorry it took so long.


Code Fragments [68K]

  Some code fragments (or snippets, if you like) that I have used.  Very limited.  Very specific to my personal code.  Probably not useful to you, but included for completeness.


Constants

  A bunch of constants.  A general mish-mash of information.  Even I don’t know why I put some of this stuff in here.


Globals by Address

  A list of all the low-memory globals that I know about.  From many sources, some accurate,  some not.  Apple says these will go away in OS X, but in the meantime they're a lot faster than using the corresponding routines.


Globals by Name

  The same information as Globals by Address, just in alphabetical order by name.


HTML Tags

  A few HTML tags for use in a text editor.


Pascal Calls

  The trap macros directly out of Inside Macintosh.  I only got up to volume IV.  Since we’ve drifted away from Pascal, I didn’t see any point in updating the information.  I was planning on a C version, but it never happened, and most probably never will.


PowerPC Mnemonics

  A list of PowerPC Mnemonics, with data.  This is the only data file that has the data in an arbitrary format that is not meant to be pasted into a document.  (I suppose you could....)  I restricted the list to the common mnemonics available on all the PowerPC processors presently being used by Apple for maximum compatibility.  If you want info on a 64-bit or a PPC 601-only mnemonic, you won’t find it here.


PowerPC Routines

  These are routines that I have used in my code.  The list is growing, so bear with me here.  Formatted for Lightsoft’s Anvil development environment and my tastes.  A few routines (marked with asterisks) are not supported by Anvil.  I tried to use them, but...


Record Definitions

  I call them records because that is the term that Inside Macintosh uses.  With offsets in hex for MacsBug use and in decimal for defining in your source code.  Sorry folks, a bunch of the networking ones are missing because I just plain got sick of typing them in.  Ditto for QuickTime.  Probably nobody will miss them anyway.


Result Codes

  As many as I could find in the volumes of Inside Macintosh that I have.


RMaker Templates

  For Consulair’s RMaker resource compiler.  Probably totally useless to you.  Or maybe not...
